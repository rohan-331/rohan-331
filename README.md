### 🟣 About Me

I'm a Computer Science undergraduate at **IIT Tirupati**, focused on building full-stack web applications, RESTful APIs, and performance-oriented software systems. I enjoy taking a project from a blank repo to something people can actually use — from grid-based canvas engines to security scanning tools to network monitors.

- 🎓 B.Tech in Computer Science Engineering, IIT Tirupati (2024 – 2028)
- 🛠️ Comfortable across the stack: React/Next.js on the frontend, Node.js/Express on the backend, with a strong foundation in DSA, OOP, and software engineering principles
- 🧩 I like designing modular, scalable architectures rather than quick hacks
- 🌱 Currently deepening my backend and systems-design skills

**Open to:** Internships and entry-level SDE roles, full-stack projects, and open-source collaboration.
